# Landing Page Web App

## Description
A landing page for "Tasty Bites" showcasing the best burgers.  
Includes a hero section with navigation links and a call-to-action button.

## Features
- Responsive hero section
- Navigation bar with hover effects
- Animated slide-in effects for text and navbar
- Call-to-action button for ordering

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Clone the repository:
git clone https://github.com/Muskan2771/Landing_Page_WebApp.git

2. Open `index.html` in any web browser.

## Learning Outcomes
- Learned CSS animations and transitions
- Practiced responsive design and flexbox layout
- Implemented a landing page UI with a hero section
